WDO para mod Harbour V2

La manera mas fácil de conectar con bases de datos con mod Harbour V2 en la Web

Descripción

WDO Web Database Objects (WDO) es una librería que te permitirá conectar con bases de datos desde mod_harbour de una manera fácil y potente. Básicamente podremos conectar a cualquier RDBMS y usar las tablas de datos de la misma manera indepedientemente de cual sea el motor de datos.
Habrá una forma particular de usar los DBF y otra de usar DB populares como MySql, Postgre, SQLite, Oracle… En esta primera versión ya tenemos disponible la conexión a Dbf y a MySql
